Cross-Chain Transfer Protocol & Arc Bridge

Native USDC transfers across blockchains with Circle's CCTP and Arc's App Kit Bridge capability

📚 Related: App Kit: Bridge | Arc Overview | Circle Index

📋 Documentation Index

Complete Arc documentation: https://docs.arc.network/llms.txt
Circle platform index: https://developers.circle.com/llms.txt

Cross-Chain Transfer Protocol (CCTP) is a permissionless onchain utility that facilitates native USDC transfers across blockchains. CCTP burns USDC on the source blockchain and mints it on the destination blockchain, enabling secure 1:1 transfers without traditional bridge liquidity pools or wrapped tokens.

💡 Tip: Use Arc App Kit's Bridge capability to simplify crosschain transfers with CCTP. Arc App Kit is an SDK that leverages CCTP as its protocol provider, letting you transfer USDC between blockchains in just a few lines of code.

Key features

💰

Native USDC transfers

Transfer native USDC across blockchains without wrapped tokens or liquidity pools

🚀

Configurable transfer speeds

Choose between Fast Transfer for speed or Standard Transfer for cost efficiency

⚙️

Programmable hooks

Trigger automated actions on the destination blockchain after USDC arrives

Arc App Kit: Bridge

Arc's App Kit Bridge is a high-level abstraction over CCTP that enables you to bridge USDC across blockchains in just a few lines of code. The Bridge capability handles the underlying burn, attestation, and mint steps automatically, making it perfect for building crosschain applications.

🚀 Quick start with Arc Bridge: Transfer 1.00 USDC from Solana to Arc Testnet with this single method call: 
const result = await kit.bridge({
  from: { adapter: solanaAdapter, chain: "Solana_Devnet" },
  to: { adapter: viemAdapter, chain: "Arc_Testnet" },
  amount: "1.00",
});

Bridge Setup & Installation

Get started with Arc App Kit Bridge by installing the necessary packages:

1. Install the Bridge package

npm install @circle-fin/bridge-kit
# or
yarn add @circle-fin/bridge-kit

2. Install adapters for your chains

Choose the adapter(s) for the blockchains you want to bridge between:

npm install @circle-fin/adapter-viem-v2 viem
# or
yarn add @circle-fin/adapter-viem-v2 viem
npm install @circle-fin/adapter-ethers-v6 ethers
# or
yarn add @circle-fin/adapter-ethers-v6 ethers
npm install @circle-fin/adapter-solana-kit solana
# or
yarn add @circle-fin/adapter-solana-kit solana
npm install @circle-fin/adapter-circle-wallets
# or
yarn add @circle-fin/adapter-circle-wallets

Bridge Quickstarts

Follow these step-by-step guides to implement specific bridge scenarios:

CCTP enables you to build applications that require moving USDC across blockchains. Here are some common use cases:

Rebalance USDC holdings across blockchains to meet liquidity demands, manage treasury positions, or take advantage of market opportunities with minimal latency.

Enable users to swap tokens on one blockchain for tokens on another blockchain by routing through USDC. Build seamless crosschain trading experiences that feel like a single transaction.

Accept USDC payments on one blockchain and automatically transfer funds to another blockchain where your business operations are based or where recipients prefer to receive funds.

Use CCTP hooks to chain together crosschain actions. Transfer USDC across blockchains and automatically deposit it into DeFi protocols, purchase NFTs, or execute smart contract logic.
